Anatomía ganadora

La estructura de un prompt que funciona

  1. 1. Rol: Decile al modelo qué tiene que ser.
  2. 2. Contexto: Explicá para quién y para qué.
  3. 3. Tarea: Pedí la acción concreta.
  4. 4. Formato: Indicá si querés HTML, tabla, lista o código.
  5. 5. Restricciones: Un archivo, mobile first, sin dependencias extra.
  6. 6. Modo: Aclarale si debe trabajar en modo agente (leyendo y editando archivos directamente).

Bloques clave

Prompts listos para copiar y pegar

Cada tarjeta explica QUÉ pedir, POR QUÉ sirve y te deja un prompt base listo para reutilizar con cualquier modelo de IA.

01 Formato

HTML autocontenido

Pedile que genere una presentación en un único archivo HTML, con todo adentro, lista para compartir y con localStorage si necesitás persistencia de datos en el equipo del usuario.

Prompt rápido

Creá una presentación en HTML autocontenido, en un único archivo, sin depender de archivos externos adicionales y compatible con cualquier navegador moderno. Si necesitás datos persistentes en el equipo del usuario, almacenalos con localStorage.

02 Responsive

Tailwind CSS

Sirve para exigir un layout mobile first, limpio, moderno y con clases utilitarias rápidas de mantener.

Prompt rápido

Usá Tailwind CSS para lograr un diseño mobile first, visualmente moderno, limpio y adaptable a celular, tablet y desktop.

03 Calidad

UX/UI

Con esto pedís una experiencia clara, usable, bien jerarquizada y con una interfaz atractiva y profesional.

Prompt rápido

Diseñá la interfaz con criterios sólidos de UX/UI: buena jerarquía visual, excelente legibilidad, navegación intuitiva y estética profesional.

04 chartjs.org

Chart.js

Pedile que use gráficos claros y presentables para mostrar datos sin complicarte con configuraciones pesadas.

Prompt rápido

Incorporá gráficos usando Chart.js para visualizar la información de forma clara, atractiva y fácil de interpretar.

05 Extracción

OCR / Extracción

Le decís al modelo que actúe como extractor de texto desde imágenes, capturas, escaneos o documentos visuales.

Prompt rápido

Actuá como un sistema OCR: extraé todo el texto visible de la imagen, respetando estructura, títulos, tablas y números.

06 Creatividad

Sorprendeme

Es la palabra para pedir creatividad, ideas frescas, composiciones visuales más ambiciosas y soluciones menos obvias.

Prompt rápido

Sorprendeme con una propuesta visual creativa, elegante y memorable, sin perder claridad, funcionalidad ni buen gusto.

07 Imágenes

Imágenes en base64

En un archivo HTML autocontenido, una imagen enlazada por ruta externa se rompe al mover o compartir el archivo. Pedile que la convierta a base64 para que quede incrustada dentro del propio HTML y siempre se vea bien.

Prompt rápido

Para que las imágenes queden bien en el archivo autocontenido, convertilas a base64 e incrustalas directamente en el HTML, sin depender de rutas ni archivos externos.

Combinaciones que funcionan

Mezclas que elevan el resultado

HTML autocontenido + Tailwind CSS + UX/UI

Ideal para landing pages, presentaciones, dashboards o demos listas para compartir. Si necesitás persistencia local, sumale localStorage.

Chart.js + UX/UI

Perfecto para reportes claros, visuales y fáciles de defender frente a equipos o clientes.

OCR + HTML autocontenido

Útil cuando primero extraés contenido de una imagen o documento escaneado y después querés convertirlo en una pieza visual bien presentada.

Plantilla base

Un prompt completo, listo para adaptar

Creá una presentación en HTML autocontenido, usando Tailwind CSS con enfoque mobile first. Aplicá principios sólidos de UX/UI para lograr una interfaz clara, profesional y visualmente atractiva. Si necesito datos persistentes en el equipo del usuario, guardalos con localStorage. Si hay datos para visualizar, incorporá gráficos con Chart.js. Si usás imágenes, convertilas a base64 e incrustalas en el HTML para que se vean bien sin depender de archivos externos. Si te comparto una imagen, actuá como OCR para extraer el texto antes de estructurarlo. Sorprendeme con una propuesta creativa, elegante y fácil de compartir.
