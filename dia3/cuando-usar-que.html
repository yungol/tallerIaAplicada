01

Chat

El punto de partida

¿Cuándo usarlo?

Cuando necesitás una respuesta rápida.

Cuando todavía estás explorando el problema.

Cuando querés hacer preguntas encadenadas para entender mejor un tema.

Ideal para arrancar, no para una investigación exhaustiva.

Idea fuerza

Chat sirve para preguntar, no necesariamente para construir una tesis completa.

02

Deep Research

Investigar con profundidad

¿Cuándo usarlo?

Cuando necesitás antecedentes, contexto y estado del arte.

Cuando querés ir más allá de una respuesta puntual.

Cuando buscás una base para un informe, una clase o una decisión.

Ideal para temas regulatorios, técnicos o comparativos.

Idea fuerza

Deep Research sirve para investigar, conectar fuentes y estructurar mejor el conocimiento.

03

Agentes

Especialistas reutilizables

¿Cuándo usarlo?

Cuando el tema se repite y querés un experto reusable.

Cuando necesitás combinar prompt + documentos + reglas de respuesta.

Cuando querés un analista especializado para un área puntual.

Ideal para compliance, tributario, legal, auditoría, riesgos.

Idea fuerza

Un agente no es «más IA». Es IA especializada para un contexto concreto.

04

Cowork

Ejecutar, no solo sugerir

¿Cuándo usarlo?

Cuando querés que la IA ejecute y no solo sugiera.

Cuando necesitás crear archivos, editar archivos o producir informes.

Cuando querés ese «amigo informático y teso en Excel» trabajando con vos.

Ideal para tareas operativas del día a día con alto componente digital.

Idea fuerza

Cowork es pasar de «decime qué hacer» a «hacelo conmigo».
